# Stylus

Take catppuccin themes and replace the mocha color palette with this:

```css
@mocha: {
    @rosewater: #f5e0dc;
    @flamingo: #f2cdcd;
    @pink: #f5c2e7;
    @mauve: #cba6f7;
    @red: #fa7970;
    @maroon: #eba0ac;
    @peach: #faa356;
    @yellow: #f9e2af;
    @green: #7ce38b;
    @teal: #94e2d5;
    @sky: #89dceb;
    @sapphire: #74c7ec;
    @blue: #77bdfb;
    @lavender: #cea5fb;
    @text: #ecf2f8;
    @subtext1: #c6cdd5;
    @subtext0: #89929b;
    @overlay2: #89929b;
    @overlay1: #21262d;
    @overlay0: #21262d;
    @surface2: #161b22;
    @surface1: #161b22;
    @surface0: #161b22;
    @base: #0d1117;
    @mantle: #0d1117;
    @crust: #0d1117;
}
```
